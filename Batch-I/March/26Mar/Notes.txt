CRUD (perform these CRUD operations in db)

Create - Create/ Insert  (Insert is to insert data into the table) 
Read  -  Select  (to query/ read the data from the table)
Update - Update (to update any value)  
Delete - Delete (to delete records from the table)


Conditional Clause

1. WHERE : to filter the entire table based on a condition

Clause

2. ORDER BY: sort the data in ascending and descending order
   ASC | DESC

3. NULL: no data is available for this column 
   

4. AND | OR : for joining multiple conditions
AND meaning all the conditions should be true or satisfied (all)
OR meaning any one of the condition is true or satisfied (any)
